STM32\+F446\+RE bare metal LED blink example. 

\begin{DoxyAuthor}{Author}
Vik-\/\+Prabhu 
\end{DoxyAuthor}
\begin{DoxyVersion}{Version}
1.\+0

\begin{DoxyDate}{Date}

This file demonstrates bare metal GPIO control on STM32\+F446\+RE Nucleo board without using HAL library. It blinks the onboard LED connected to PA5 using direct register access.

\begin{DoxyNote}{Note}
Hardware:\+ STM32\+F446\+RE Nucleo-\/64 board LED :\+ PA5 (onboard green LED) Clock :\+ Internal HSI 16MHz (default after reset)
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Do not enable HAL library alongside this code. This is pure bare metal register level programming. 
\end{DoxyWarning}

Initializes PA5 as a push-\/pull output for LED control. 

Performs the following steps:\+
\begin{DoxyEnumerate}
\item Enable GPIOA clock via RCC AHB1\+ENR
\item Set PA5 MODER bits to output mode (01)
\item Set PA5 OTYPER to push-\/pull (0)
\item Set PA5 OSPEEDR to low speed (00)
\item Set PA5 PUPDR to no pull (00)
\end{DoxyEnumerate}
