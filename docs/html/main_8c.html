Detailed Description

STM32F446RE bare metal LED blink example.

Author
Vik-Prabhu
Version
1.0
Date
2024-01-01

This file demonstrates bare metal GPIO control on STM32F446RE Nucleo board without using HAL library. It blinks the onboard LED connected to PA5 using direct register access.

Note
Hardware: STM32F446RE Nucleo-64 board LED : PA5 (onboard green LED) Clock : Internal HSI 16MHz (default after reset)
Warning
Do not enable HAL library alongside this code. This is pure bare metal register level programming.

◆ LED_Init()

void LED_Init ( void )

Initializes PA5 as a push-pull output for LED control.

Performs the following steps:

  1. Enable GPIOA clock via RCC AHB1ENR
  2. Set PA5 MODER bits to output mode (01)
  3. Set PA5 OTYPER to push-pull (0)
  4. Set PA5 OSPEEDR to low speed (00)
  5. Set PA5 PUPDR to no pull (00)
Note
PA5 is the onboard green LED on Nucleo-F446RE board

void LED_Toggle ( void )

Toggles the state of onboard LED on PA5.

Reads the current state of PA5 from ODR and flips it using XOR operation. This is safer than read-modify-write on IDR.

Note

int main ( void )

Main entry point of the application.

Initializes the LED pin and blinks it forever in an infinite loop. Execution flow:

  1. Initialize PA5 as output
  2. Enter infinite loop
  3. Toggle LED
  4. Wait using software delay
  5. Repeat
